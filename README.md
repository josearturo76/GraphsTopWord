# GraphsTopWord

- Projeto consiste em mostrar as 10 palavras mais usadas em sites plotadas através de gráficos

- Ao abrir a pagina basta digitar a URL no campo existente e clicar nas opções de gráfico Column Chart, Pie Chart ou Line Chart.

- Projeto desenvolvido em ASP.NET MVC 5 em C#

- .Net Framework 4.5

- Google Charts para gerar graficos

- HTMLAgilityPack (Nuget) versão 1.11.33, para converter site em texto puro e capturar as palavras
